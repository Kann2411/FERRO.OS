# FERRO.OS AI Agent

Esta carpeta contiene las instrucciones oficiales para los agentes de IA que participarán en el desarrollo de FERRO.OS.

Cada archivo representa una fase del Roadmap.

Antes de comenzar una fase, el agente debe leer únicamente la documentación necesaria indicada en el archivo correspondiente.

Esto reduce consumo de contexto y mantiene el desarrollo consistente.

Cada fase incluye:

- Objetivo
- Documentación obligatoria
- Milestones
- Restricciones
- Criterios de aceptación
- Prompt listo para ejecutar

Nunca debe saltarse una fase.

Cada milestone debe quedar funcionando antes de continuar.

FERRO.OS se construye incrementalmente.
